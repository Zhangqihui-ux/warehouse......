# WarehousePro Inventory API
A RESTful API for inventory management, built with Node.js, Express, and MongoDB native driver (for Web Data Management Assignment 1).

## Prerequisites
- Node.js (v14+ recommended)
- npm (v6+ included with Node.js)
- MongoDB Atlas account (free tier works)

## Local Installation (VS Code)
1. Clone this private GitHub repository to your local machine.
2. Open the project folder in VS Code.
